#`TwitDuke-Core` Module

TwitDukeに密接に関係するコア機能を集めたモジュールです。


###`TwitDuke-Core`で提供される主なクラス
 * アカウントの読み/書き  
`AccountManager`など
 * Twitter OAuth  
`PINAuthentication`,`LambdaOAuth`など
 * キー設定済みTwitter関連オブジェクト  
 `TAsyncTwitterFactory`など
 * I/O  
 `Reader<T>`,`Writer<T>`など
 * ラッパークラス  
 `Twitter`,`Port`など
