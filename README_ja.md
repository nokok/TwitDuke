#TwitDukeとは
##### TwitDukeは開発中のソフトウェアです。まともに動きません
[![Build Status](https://travis-ci.org/nokok/TwitDuke.svg?branch=master)](https://travis-ci.org/nokok/TwitDuke)  
TwitDukeはTwitterクライアントです

#ビルド

    % git clone https://github.com/nokok/TwitDuke
    % cd TwitDuke
    % mvn package

#バイナリ
